# Game of Life
 
OOP implementation of famous Game of Life

### Target platform: Ubuntu 16.04

Language: C++11 compiled under GCC 5.4.0 using CMake 3.5.2

## Installing

In bash:

```
    cmake ./CMakeLists.txt
    make
```
